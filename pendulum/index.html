Magnetic pendulum

A pendulum swings over three magnets — a spring pull toward the center, a little friction, inverse-square attraction to each magnet. Every pixel is a release point, colored by the magnet that finally captures the pendulum. The system is fully deterministic, but near a basin boundary the outcome flips with arbitrarily small changes in the release point, so the boundary is fractal. Scroll to zoom, drag to pan, hover to trace a trajectory, click to pin it.

Rendered progressively across Web Workers, one full simulation per pixel. "Shade by capture time" steps each basin through discrete lightness bands — light means the pendulum settles quickly, and each darker band is a contour of longer wandering, so the darkest filigree hugs the boundary saddles. The integrator is fixed-step symplectic Euler, so at extreme zoom you are looking at the basin structure of that discrete map rather than of the exact continuous flow.
